# onevarez — Claude Code Plugins

A marketplace of plugins for [Claude Code](https://claude.ai/code).

```bash
claude plugins marketplace add onevarez/claude-plugins
```

## Plugins

| Plugin | Description | Install |
|---|---|---|
| [Viewfinder](plugins/viewfinder/docs/README.md) | Cinematic browser session recording with auto-zoom | `claude plugins install viewfinder@onevarez` |
